简介：
	这是一个简单的STM32F103C8T6实现万年历的程序，辅以PWM控制的呼吸灯光效。

接线方式：
	将OLED的SCL引脚接在PB8，SDA接在PB9，
	按键接在PA4，按键按下输入低电平，
	旋转编码器接在PA6、PA7。

使用方式：
	上电后，时钟从1970年1月1日0时0分0秒开始计时，
	按下按键，进入调节时间模式，通过旋转编码器依次调节年、月、日、时、分、秒；
	每项调节完成，按下按键确认。
	全部调节完成后，时钟从调节到的时刻开始计时。

注意事项：
	日期与时间的显示由RTC驱动，因此在有备用电源供电情况下，主机断电时时钟仍会走动；
	呼吸灯的亮暗周期时长为2秒。
